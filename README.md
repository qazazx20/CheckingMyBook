# CheckingMyBook

* 웹으로 읽은 책을 기록할 수 있는 사이트가 적어 선정
* 스프링 서적을 기반으로 CRUD에 기초한 웹 애플리케이션

## 개발 환경
* IDE: Spring Tool Suite 3.9.18
* Language: Java
* Database: Oracle
* Framework: Spring, MyBatis
* Server: Apache-Tomcat 8.5

* Web: HTML, CSS, JQuery, JSP

## 구현 기능
* 회원가입, 로그인, 로그아웃
* 책 관련 게시글 생성, 조회, 변경, 삭제
